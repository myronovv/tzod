# tzod
Технології збору та обробки даних

**Практична робота №1. Агрегація та аналіз даних**

У цій роботі було створено таблицю з даними про товари та суми продажів. За допомогою бібліотеки Pandas виконано групування даних за назвою товару та розраховано основні статистичні показники: кількість продажів, загальну суму, середнє значення, медіану та стандартне відхилення.

Додатково було визначено частку кожного товару в загальному доході. Для зручності аналізу сформовано підсумкову таблицю, яка містить топ-5 товарів за сумою продажів, а всі інші товари об’єднані в окрему категорію «Інші».

Результат: програма дозволяє швидко оцінити найбільш прибуткові товари та їхній внесок у загальну виручку.

**Практична робота №3. Оптимізація використання пам’яті в DataFrame**

У роботі було створено великий набір даних на 100 000 записів, що містить інформацію про регіони, типи енергії, обсяги споживання та виробництва, відсоток втрат і рік.

Після формування таблиці виконано аналіз обсягу пам’яті, яку вона займає. Далі проведено оптимізацію типів даних:

текстові поля перетворено на тип category;
числові значення зменшено до типів int16;
дробові значення переведено у float32.

Після оптимізації було повторно обчислено використання пам’яті та визначено, скільки мегабайт і відсотків вдалося зекономити.

Результат: практична робота показує, що правильний вибір типів даних значно зменшує обсяг пам’яті та робить обробку великих таблиць ефективнішою.

**Практична робота №4. Відновлення пропущених значень методом інтерполяції**

У цій роботі було змодельовано часовий ряд енергоспоживання за 200 днів. Частину значень штучно видалено, щоб перевірити різні способи відновлення пропусків.

Для заповнення відсутніх даних використано три методи інтерполяції:

лінійна;
поліноміальна;
сплайн-інтерполяція.

Якість кожного методу оцінено за допомогою метрики MSE — середньоквадратичної похибки. Також побудовано графіки, які візуально порівнюють початкові значення, пропуски та результати відновлення.

Результат: робота дозволяє порівняти точність різних методів інтерполяції та визначити, який із них краще підходить для відновлення часових даних.

**Практична робота №5. Заповнення пропущених значень у наборі даних**

У роботі розглянуто простий набір даних із пропущеними значеннями споживання. Було реалізовано кілька окремих функцій для заповнення пропусків різними способами:

середнім значенням;
медіаною;
попереднім відомим значенням (ffill);
наступним відомим значенням (bfill).

Кожен метод застосовується до копії початкової таблиці, після чого результат виводиться окремо для порівняння.

Результат: практична робота демонструє основні підходи до обробки пропущених значень і дозволяє побачити різницю між ними на конкретному прикладі.

**Практична робота №6. Порівняння рівня освітленості різних джерел світла**

У цій роботі аналізуються дані з трьох CSV-файлів, що відповідають різним типам ламп:

LED;
Fluorescent;
Incandescent.

Для кожного джерела світла обчислюється середнє значення яскравості за показником Brightness EV. Також побудовано:

лінійний графік зміни рівня освітленості в часі;
стовпчикову діаграму для порівняння середньої яскравості.

Якщо у вхідних даних відсутній стовпець часу, програма автоматично створює його на основі порядкового номера запису.

Результат: програма дозволяє наочно порівняти стабільність і середній рівень освітленості різних типів ламп.
